## Luno Premium 

<h3>Introduction</h3>
a command line application that console logs out the price difference and percentage between Luno and Binance for different cyptocurrency.

<h3>Develop Environment</h3>
Javascript, Nodejs, BinanceAPI, Luno API

<h3>Installation</h3>
Before running the script, please 
<br />
<br />
<ol>1. Install node.js on your local machine.</ol>
<ol>2. Set APILAYER_KEY with your fixer API key.</ol>
<ol>You may setup the key at https://apilayer.com/marketplace/fixer-api and store the APILAYER_KEY in .env file.</ol>
<ol>3. Clone the repository or download the script file.</ol>
<ol>4. Run npm install to install required dependencies.</ol>

<h3>Usage</h3>
<ol>1. Open a terminal and navigate to the script directory.</ol>
<ol>2. Run the script using the command <strong>node index.cjs</strong>.</ol>
<ol>3. Enter the target cryptocurrency when prompted (e.g., "BTC," "ETH").</ol>
<ol>4. View the displayed information, including prices on Binance and Luno, price difference, and Luno premium.</ol>

<h3>Important Note</h3>
Ensure that you keep your API keys secure and do not share them. The script relies on external APIs, and any changes to these APIs may impact its functionality.
